# トモスマHP設定

## 構成
- index.html：トップページ
- news.html：ニュース・活動報告
- events.html：イベント情報
- data.js：動的データ（ここを主に編集）
- style.css：スタイル（テーマカラー：リビングコーラル）

## 更新ルール
- テキスト・データ変更は data.js を編集
- 編集後は必ず git add . && git commit -m "更新内容" && git push する
- pushするとVercelが自動デプロイする

## 禁止事項
- デザインの大幅変更は渡辺さんの承認が必要
- 画像の追加・削除は確認してから
